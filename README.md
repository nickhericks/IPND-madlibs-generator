# IPND-madlibs-generator
A madlibs generator written in python (part of Udacity's Intro to Programming Nanodegree program)

Run the python program in the command line to play the game.

Answer key:

Easy level
1. world
2. python
3. print
4. html

Medium level
1. function
2. arguments
3. none
4. list

Hard level
1. class
2. method
3. __init__
4. instance
